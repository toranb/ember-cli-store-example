This project shows the ember-cli addon [Ember-Cli-Simple-Store][] in action

## Run the app in your browser

To run the app in your browser

```
1) npm install
2) bower install
3) ember server
```

## Run the tests

```
1) npm install
2) bower install
3) ember test
```

[Ember-Cli-Simple-Store]: https://github.com/toranb/ember-cli-simple-store
